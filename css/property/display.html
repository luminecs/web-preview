display 属性

此属性用于控制元素的展示方式。

CSS 语法

display: inline|block|contents|flex|grid|inline-block|inline-flex
  |inline-grid|inline-table|list-item|run-in|table|table-caption
  |table-column-group|table-header-group|table-footer-group
  |table-row-group|table-cell|table-column|table-row
  |none|initial|inherit;

JS 语法

document.getElementById('d1').style.display = 'none'

示例1 display: none|inline|block|inline-block|list-item
