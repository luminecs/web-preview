flex 属性

flex 是 flex-grow、flex-shrink、flex-basis 的简写版本， 用于设置 flex 布局。父元素盒子必须设置 display: flex;，flex 才能生效。
默认值是 0 1 auto。

CSS 语法

flex: flex-grow flex-shrink flex-basis|auto|initial|inherit;

JS 语法

document.getElementById('d1').style.flex = '1'
