flex-shrink 属性

此属性设置当前元素相对于(相同盒子里)其他元素收缩多少。 父元素盒子必须设置 display: flex;，flex-shrink 才能生效。 flex-shrink 表示的是缩小倍数。
默认值是 1。

CSS 语法

flex-shrink: 数字|initial|inherit;

JS 语法

document.getElementById('flex-shrink').style.flexShrink = '5'

示例1

flex-shrink 需要和 flex-basis 一起使用。 如果只设置 flex-shrink: 1; ，则无效果。flex-grow 属性可选。

如果子元素 flex-basis 之和小于父元素的宽度，则 flex-shrink 无效果。
