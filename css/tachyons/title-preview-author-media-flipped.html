笔记

绝对长度：px in cm

  • px: 像素
  • in: 英寸 1inch = 96px
  • cm: 1cm = 37.8px = 25.2/64in

相对长度：em rem vw

优先使用相对长度，尤其是 font-size

  • em 在处理 font-size 时，与父元素的字体大小有关； 在处理其他属性，例如 height 时，与元素自身字体大小有关
  • rem 是基于根元素的 font-size 计算得来的。

    说明： 

        html { // 或者通过伪类 :root
      font-size: 18px; // 默认值是 16px
    }
    p {
      max-width: 2rem; // 2 * 18px = 36px
    }

    方便计算： 

        html {
      font-size: 62.5%; // 16px * 0.625 = 10px
    }
    p {
      max-width: 1.8rem; // 1.8 * 10px = 18px
    }

  • vw 表示窗口宽度的 1%，如果元素 width = 10vw，则元素宽度 = 窗口宽度的 10%
