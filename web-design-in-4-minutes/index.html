四分钟入门网页设计

by luminecs

你有一个产品，或者一个想法，现在你想要在自己的网站上展示并分享给别人。在发布到互联网之前，你想让它看起来很有吸引力、很专业，或者至少看起来是体面的。

你首先应该做什么呢？

内容

设计的目的在于增强内容的展示。这听起来好像是显而易见的，但是作为网站最主要的组成部分--内容，它不应该建立在事后的想法上，即应该提前考虑内容的展示。

编写内容，就像你现在正在阅读的段落一样，组成整个互联网的 90% 的部分。给这样的文本内容调整样式将会占用很多时间。

假设你已经完成了你想要发布的内容，并且也已经创建了一个空的 style.css 文件，你应该写的第一个样式是什么呢？

居中

多行文本很难被转换，因而难以阅读。设置每行的字符数限制将会大大地增强可读性、提升文字的吸引力。 

body {
  margin: 0 auto;
  max-width: 50em;
}

在给文本块设置样式之后，如何给文字本身设置样式呢？

字体（Font Family）

浏览器默认字体是 "Times"，看起来没有一点吸引力，主要是因为这是一种未经样式修改（unstyled）的字体。切换到类似 "Helvetica" 或 "Arial" 的有衬线字体（sans-serif），将极大地提升网页视觉。 

body {
  font-family: "Helvetica", "Arial", sans-serif;
}
